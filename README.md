# CADENZA · ברושור דיגיטלי

ברושור אינטראקטיבי של **CADENZA Piano – פסנתר רחוב בע"מ**, להצגת שיתוף פעולה עם רשויות מקומיות והצבת פסנתרי רחוב במרחב הציבורי.

## 🔗 קישור חי

לאחר הפעלת GitHub Pages, הברושור האינטראקטיבי זמין בכתובת:

**https://office644.github.io/cadenza-brochure/**

זוהי הגרסה המומלצת לשליחה לראשי ערים — לחיצה, ניווט, קרוסלה, פתיחת יתרונות והגדלת תמונות עובדים במלואם, וגם בנייד.

## תוכן המאגר

| קובץ | תיאור |
|------|--------|
| `index.html` | הברושור האינטראקטיבי (קובץ ההצגה הראשי) |
| `cadenza-qr.png` | קוד QR לאתר החברה (חייב להישאר לצד `index.html`) |
| `CADENZA-Brochure.pdf` | גרסת PDF להדפסה / צירוף למייל (ניווט פנימי לחיץ) |
| `docs/CADENZA-Email.docx` | נוסח מייל פנייה לרשות |
| `docs/CADENZA-Letter.docx` | מכתב רשמי לפנייה לרשות |
| `docs/CADENZA-Proposal.docx` | דף הצעה לשיתוף פעולה |

## עריכה מהירה

- **פרטי קשר / טלפון / קישור יומן** — בתוך `index.html`, חפשו `const CONFIG`.
- **תמונות** — תמונת ה-Hero, הקרוסלה והגלריה נמשכות מאתר החברה. ניתן להחליף לכתובות/קבצים אחרים.
- **חלוקת גלריה לפי ערים** — אפשר להוסיף סינון לפי ערים לאחר שיוך כל תמונה למיקומה.
